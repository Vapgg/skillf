Инструкция по работе с Git

Для выполнения данной работы вам понадобится установить Git на ваш компьютер. Вы можете скачать его с официального сайта git-scm.com.

Основные операции Git

Для выполнения основных операций с Git, ознакомьтесь с видеозаписями:

  • Видео 1: Установка Git
  • Видео 2: Создание репозитория
  • Видео 3: Добавление файлов в репозиторий
  • Видео 4: Коммит изменений
  • Видео 5: Отправка изменений на удаленный репозиторий
  • Видео 6: Получение изменений из удаленного репозитория

Основные команды Git

Основные команды Git:

        
            git init
            git add <файлы>
            git commit -m "Сообщение коммита"
            git push origin <ветка>
            git pull origin <ветка>

Работа с репозиторием

Для работы с репозиторием, следуйте инструкциям в соответствующих видеозаписях.
